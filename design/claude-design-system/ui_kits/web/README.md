# Web UI kit — Daily Cadence

Desktop companion to the mobile app. Sidebar-driven dashboard with richer analytics.

## Screens
1. **Dashboard** — Today overview with widgets, recent timeline, mood chart.
2. **Analytics** — Multi-chart view for progress metrics over time.

## Files
- `index.html` — interactive prototype with sidebar nav
- `components.jsx` — shared primitives (Sidebar, Topbar, Card, Chart helpers)
- `Dashboard.jsx`, `Analytics.jsx`
