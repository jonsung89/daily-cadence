# Daily Cadence 🎵

A beautiful day logger inspired by Google Keep with built-in progress tracking.

## Vision

Log your workouts, meals, sleep, mood, and activities throughout your day on a clean timeline. See patterns and progress automatically with fully customizable themes.

## Features (Coming)

- Daily timeline view (iOS first)
- Workout tracking with progress visualization
- Meal logging with optional macro tracking
- Sleep and mood tracking
- Customizable note types with colors, icons, and themes
- Progress charts and calendar views
- Customizable dashboard widgets

## Tech Stack

- **iOS:** Swift + SwiftUI
- **Backend:** Firebase (Firestore, Cloud Functions)
- **Design System:** Custom (Anthropic Academy inspired)
